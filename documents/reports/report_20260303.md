### 2. MLflow Backend Migrated to PostgreSQL

**File changed**: `mlflow-research/docker-compose.yml`

Migrated the MLflow backend store from SQLite to PostgreSQL for better robustness and concurrent-write safety.

#### Changes to `docker-compose.yml`

| | Before | After |
|---|---|---|
| Backend store | `sqlite:////home/mlflow/mlruns/mlflow.db` | `postgresql://mlflow:mlflow@postgres:5432/mlflow` |
| DB container | — | `postgres:16-alpine` |
| DB persistence | Host file `./mlruns/mlflow.db` | Named volume `postgres_data` |
| MLflow volume | `./mlruns:/home/mlflow/mlruns` | `./mlruns/artifacts:/home/mlflow/mlruns/artifacts` |
| Startup order | — | `depends_on: postgres: condition: service_healthy` |

**Added PostgreSQL service:**
```yaml
postgres:
  image: postgres:16-alpine
  container_name: mlflow-postgres
  environment:
    POSTGRES_USER: mlflow
    POSTGRES_PASSWORD: mlflow
    POSTGRES_DB: mlflow
  volumes:
    - postgres_data:/var/lib/postgresql/data
  healthcheck:
    test: ["CMD-SHELL", "pg_isready -U mlflow -d mlflow"]
```

**No changes to `Dockerfile`** — `psycopg2-binary` was already installed.
